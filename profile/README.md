## What we do

- **Conversational & Voice Agents**: support, routing, appointment booking, lead capture  
- **Automations**: email processing, document flows, CRM/helpdesk integration  
- **Recommendations & Data AI**: personalization, scoring, analysis  
- **Custom Projects**: internal tools, APIs, SaaS, infra, MCP servers

## How we work

1. **Discovery** , map the workflow and success metrics  
2. **Prototype** , validate fast on real data  
3. **Production** , ship a robust solution (monitoring, security, maintainability)

## Open source on this GitHub

You'll find:
- templates and starter kits for **agents, RAG, evals, and automations**
- small **reusable components** we extract from client work
- experiments and demos that turn into production patterns

> Most client code lives in private repos, we open-source what's broadly useful.
